# Advanced E-Commerce Analytics Engine

## Project Overview
This project demonstrates advanced SQL techniques using a PostgreSQL database for an e-commerce business. It analyzes customer behavior, sales performance, product trends, payment transactions, and marketing attribution to generate business insights through SQL and Tableau.

## Database Schema
The solution uses a normalized 6-table relational database:

- Customers
- Products
- Orders
- Order Items
- Payment Ledger
- Marketing Attribution

The schema uses primary keys, foreign keys, and constraints to maintain data integrity.

## SQL Analytics
The project includes 25 business-focused SQL queries covering:

- Monthly Sales Trends
- Customer Lifetime Value (CLV)
- Cohort Retention
- Marketing Attribution
- Average Order Value
- Repeat Purchase Analysis
- Payment Performance
- Product Category Analysis
- Customer Segmentation
- Revenue and Operational KPIs

The queries demonstrate joins, CTEs, window functions, ranking functions, subqueries, CASE expressions, and aggregate functions.

## Dashboard
A Tableau dashboard visualizes the key business KPIs generated from the SQL analysis.
